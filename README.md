# DBBP - Društvo Bistroumnih Bitij Pan-galaksije

Official website for DBBP (Society of Bright Beings of the Pan-Galaxy), a Slovenian association inspired by The Hitchhiker's Guide to the Galaxy.

## About the Project

This is a bilingual (English/Slovenian) static website built with **pure HTML5 and CSS3 only** - no JavaScript, no frameworks, no external dependencies. The design embraces a retro-futuristic terminal aesthetic inspired by The Hitchhiker's Guide to the Galaxy.

## Design Philosophy

- **Pure HTML5 + CSS3**: Zero JavaScript, zero frameworks
- **Terminal Aesthetic**: Black background (#000), green text (#0f0), yellow accents (#ff0)
- **Monospace Typography**: Courier New for that classic computer terminal feel
- **Responsive Design**: Mobile-first approach with CSS-only interactions
- **Accessibility First**: Semantic HTML5, high contrast, keyboard navigation

## Key Features

- **Bilingual Support**: Complete English and Slovenian versions
- **Language Toggle**: Switch between languages on any page
- **CSS-Only Mobile Menu**: Hamburger menu using checkbox hack
- **Sticky Header**: Navigation stays accessible while scrolling
- **Footer Navigation**: Quick links on every page
- **Responsive Layout**: Works seamlessly on desktop, tablet, and mobile

## Technical Constraints

**Allowed:**
- HTML5 semantic elements
- CSS3 (animations, transitions, flexbox, grid)
- CSS-only interactive techniques (checkbox hacks, :hover, :target, etc.)

**Not Allowed:**
- JavaScript or any scripting
- External frameworks (Bootstrap, Tailwind, React, etc.)
- CSS preprocessors (Sass, Less, etc.)
- External libraries or CDNs

## About DBBP

DBBP is a horizontal network focused on social reforms in Slovenia, advocating for:
- Basic income (Temeljni dohodek)
- 30-hour work week
- Energy and water self-sufficiency
- Housing reform
- Substance reform and mental health initiatives
- Criminal justice reform
- Economic growth and Balkan cooperation
- Human rights and ethical practices

## Development

The site is designed to work in all modern browsers (Chrome, Firefox, Safari, Edge) without any build process. Simply open any HTML file in a browser to view.
